## Overview

### Blog Preview Card 📝

A clean, modern, and responsive Blog Preview Card component built with semantic HTML5 and custom CSS3. This project is a perfect example of a foundational web component using a "Neo-brutalism" design style.

### 🚀 About the Project

This component is designed to show a summary of a blog post, including a category tag, publication date, title, description, and author profile. It focuses on clean typography and high-contrast shadows to create a bold, modern look.

### ✨ Key Features

- Neo-brutalist Design: Features sharp borders and distinctive box-shadows (5px 5px, 10px 10px) for a "pop" effect.
- Fully Responsive: The card adjusts its width to 80% on screens smaller than 450px, ensuring a great mobile experience.
- Centered Layout: Uses CSS Flexbox on the body to keep the card perfectly centered regardless of screen size.
- Optimized Typography: Carefully chosen line heights (24px for description) and font weights for better readability.
- Vibrant yellow palette

### 🛠️ How to Use

- Clone this repository to your local machine.
- Open the index.html file in any modern web browser.
- Ensure the images are placed in the ./images/ directory as referenced in the HTML.

### 📄 Component Structure

The card consists of the following sections:

- Image Header: A top-level preview image with rounded corners.
- Category Tag: A highlighted "Learning" badge.
- Content: Includes the publication date, a bold title, and a descriptive summary.
- Author Profile: A footer section containing the author's avatar and name ("Greg Hooper").
